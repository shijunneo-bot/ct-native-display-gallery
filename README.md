# Native Display Gallery — CleverTap

An interactive reference gallery showcasing **180 Native Display use cases** across 15 industry verticals, built to demonstrate the capabilities of [CleverTap's](https://clevertap.com) Native Display channel on both web and app.

🔗 **Live demo:** [View Gallery](https://shijunneo-bot.github.io/ct-native-display-gallery/)

---

## What's inside

| Section | Count | Description |
|--------|-----------|-------------|
| Use cases | 180 | 15 verticals × 12 use cases, web and app, wireframe + branded execution per card |
| Templates covered | 11 | Simple Message, Carousel, Message with Icon, Custom Key-Value, Banner, Banner Carousel, Banner with Text, image-only variants |
| Surfaces | 2 | App (iOS/Android) and Web |
| Visual Editor showcase | 8 | Before/after capability pairs plus an honest limitations list |
| FAQ | 20 | The questions clients actually ask, with straight answers |

## Features

- **Enter cover** — an animated inflated-banner cover with day/night toggle before you enter
- **Deep-dive modal** per card — 5W1H, copyable key-value payload, generated code per surface and template, why it works, KPI chips, large mockups with labels and tags
- **Filters** — by vertical, surface, mechanic, and template
- **Visual Editor showcase** — what it can and cannot do, shown as before/after
- **Developer docs section** — links to every official CleverTap page, grouped by app and web
- **Day / night themes** with measured WCAG AA contrast in both
- **Field Guide Family** — links to all twelve galleries in the network

## Usage

Every use case is a client-facing illustration built with self-contained HTML and vanilla JS. It shows what Native Display *looks and feels like* — it is not live CleverTap. For production, verify SDK versions and plan packaging against [docs.clevertap.com/docs/native-display](https://docs.clevertap.com/docs/native-display).

All brands shown (Lumo Mobile, Vela+, Arca Bank, Sagana Care, Tokobell, MakanGo, SwiftRide, Kitefare, Nebula Arena, BrightPath, Ovia Health, FreshCart, The Daily Lens, OrbitPoints, HavenList) are fictional.
